# VAC-MD 聊天框Markdown支持

基于vue-advanced-chat (opens new window)，添加完整Markdown支持

# 基本用法

与原版使用方式相同，带来

  • 更完整的Markdown格式支持
  • 代码语法高亮
  • 更宽的消息泡泡
  • 输入禁用控制
  • 聊天标题隐藏控制
  • 样式调整
