# HierarchyV 可视化指标体系编辑器

基于JointJS免费版的可视化层次指标体系编辑组件

# 基本用法

通过查看、添加节点与添加连线模式，进行拖拽布局和节点与连线的创建，构建层次指标体系。内建功能：

  • 框选节点与连线，进行批量复制粘贴、删除与移动
  • 鼠标滚轮缩放，中键拖拽移动画布（关于触控板）
  • 缩放、自动布局、撤消重做等常用操作
  • 子节点折叠与展开
  • 右键菜单快捷选项
模式切换
菜单方法

在外部通过组件方法changeState切换模式，组件通过state-change事件修改模式。通过组件属性设置功能启用，其余操作均通过组件方法调用

<template>
  <div>
    <span class="demonstration">模式切换</span>
    <el-button-group class="demonstration">
      <el-button :type="primaryButton[0]" @click="view">查看模式</el-button>
      <el-button :type="primaryButton[1]" @click="addNode">添加节点</el-button>
      <el-button :type="primaryButton[2]" @click="addLink">添加连线</el-button>
    </el-button-group>
    <span class="demonstration">菜单方法</span>
    <el-button-group class="demonstration">
      <el-button @click="deleteCell">删除</el-button>
      <el-button @click="zoom('in')">放大</el-button>
      <el-button @click="zoom('out')">缩小</el-button>
      <el-button @click="undo">撤销</el-button>
      <el-button @click="redo">重做</el-button>
      <el-button @click="autoLayout">自动布局</el-button>
    </el-button-group>
    <HierarchyV
      ref="hierarchyV"
      interactive
      context-menu
      show-expand
      @state-change="paperState=$event"
      style="height: 500px;"
    />
  </div>
</template>
<script>
  
export default {
  data() {
    return {
      paperState: null
    }
  },
  computed: {
    primaryButton() {
      return ['view', 'addHRect', 'addLink'].map(item => {
        return this.paperState === item ? 'primary' : ''
      })
    }
  },
  mounted() {
    this.$refs.hierarchyV.loadJson(this.$hierarchyVExample.basic)
  },
  methods: {
    addLink() {
      this.$refs.hierarchyV.changeState('addLink')
    },
    addNode() {
      this.$refs.hierarchyV.changeState('addHRect')
    },
    view() {
      this.$refs.hierarchyV.changeState('view')
    },
    deleteCell() {
      this.$refs.hierarchyV.deleteCell()
    },
    zoom(type) {
      this.$refs.hierarchyV.zoom(type)
    },
    undo() {
      this.$refs.hierarchyV.undo()
    },
    redo() {
      this.$refs.hierarchyV.redo()
    },
    autoLayout() {
      this.$refs.hierarchyV.autoLayout()
    }
  }
}
</script>

# 触控板支持

使用触控板双指拖拽移动画布

触控板双指缩放默认启用，此选项只控制拖拽

实验性功能：对于不同型号设备可能存在兼容性问题

通过trackpad启用触控板支持

<template>
  <HierarchyV

  />
</template>

# 导出与导入

支持导出JointJS原始格式、XML、TreeNode树形数据结构与图片；支持从JointJS原始格式导入

通过loadJson反序列化JointJS原始格式，通过toJson获取JointJS原始格式，通过toXml获取XML，通过toTree获取TreeNode，通过toImg导出图片

<template>
  <div>
    <el-button-group  class="demonstration">
      <el-button @click="toJson">序列化(JointJS)</el-button>
      <el-button @click="toXml">序列化(XML)</el-button>
      <el-button @click="loadJson">反序列化(重置)</el-button>
      <el-button @click="toTree">转换(TreeNode)</el-button>
      <el-button @click="toImg">导出图片</el-button>
    </el-button-group>
    <HierarchyV ref="hierarchyV" style="height: 500px;" />
  </div>
</template>
<script>

export default {
  mounted() {
    this.$refs.hierarchyV.loadJson(this.$hierarchyVExample.basic)
  },
  methods: {
    toJson() {
      this.$message.success('请在控制台查看数据')
      console.log(this.$refs.hierarchyV.toJson())
    },
    toXml() {
      const xml = this.$refs.hierarchyV.toXml()
      if (xml) {
        this.$message.success('请在控制台查看数据')
        console.log(xml)
      } else {
        this.$message.warning('XML序列化要求具有唯一根节点')
      }
    },
    loadJson() {
      this.$refs.hierarchyV.loadJson(this.$hierarchyVExample.basic)
    },
    toTree() {
      this.$message.success('请在控制台查看数据')
      console.log(this.$refs.hierarchyV.toTree())
    },
    toImg() {
      this.$refs.hierarchyV.toImg(true, '指标体系', 'PNG')
    }
  }
}
</script>
