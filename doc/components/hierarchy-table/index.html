# HierarchyTable 指标体系编辑表格

横向树形层次指标体系编辑表格

HierarchyTree和HierarchyV是类似功能的不同实现方式，比本组件更加简单易操作

# 基本用法

传入初始（或保存的）指标体系数据，通过增加或减少层次与创建节点实现树形指标体系的编辑

由于编辑方式的限制:

  1. 可能需要对节点进行灵活地删除和添加来得到特定树结构
  2. 如果叶节点需要不一样的层次，可通过节点名称留空表示节点不存在。这需要在业务逻辑中处理

通过init-data设置表格的初始数据，通过increaseDepth和decreaseDepth增加或减少层次

  <div>
    <el-button @click="$refs.table.increaseDepth()">增加层次</el-button>
    <el-button @click="$refs.table.decreaseDepth()">减少层次</el-button>
    <el-button @click="logData">获取数据</el-button>
    <HierarchyTable
      ref="table"
      :init-data="initData"
      style="margin-top: 10px;"
    />
  </div>
</template>
