# TextEllipsis 文本省略

按宽度自动显示省略号与Tooltip的文本容器

# 基本用法

在需要动态改变单元格内容的表格列中，无法利用表格组件自带的省略号与tooltip显示功能。使用该组件可以实现类似的效果

通过v-model绑定选中（处于输入框内）的标签，通过existing-tags设置可选的标签

<template>
  <div>
    <el-button @click="switchEdit">开启/关闭编辑</el-button>
    <el-table
      :data="tableData"
      border
    >
      <el-table-column
        prop="name"
        label="姓名"
        width="180"
      >
        <template slot-scope="scope">
          <div v-if="editing">
            <el-input v-model="scope.row.name"></el-input>
          </div>
          <div v-else>
            <TextEllipsis :content="scope.row.name" :tooltip="scope.row.name" />
          </div>
        </template>
      </el-table-column>
      <el-table-column
        prop="tel"
        label="电话"
      >
        <template slot-scope="scope">
          <div style="display: flex; width: 100%">
            <div v-show="editing">
              <el-input v-model="scope.row.tel"></el-input>
            </div>
            <div v-show="!editing" style="flex-grow: 1; width: 0">
              <TextEllipsis :content="scope.row.tel" :tooltip="scope.row.tel" />
            </div>
          </div>
        </template>
      </el-table-column>
      <el-table-column
        prop="tel"
        label="el-table自带的效果"
        show-overflow-tooltip
      >
      </el-table-column>
    </el-table>
  </div>
</template>
<script>
  
export default {
  data() {
    return {
      tableData: [
        {
          name: '张三',
          tel: '135123412341111111111111111111111111111111111111111111'
        },
        {
          name: '李四',
          tel: '135123412341111111111111111111111111111111111111111111'
        }
      ],
      editing: false
    }
  },
  methods: {
    switchEdit() {
        this.editing = !this.editing
    }
  }
}
</script>
